## 不实现（远期或放弃）

| 功能 | 原因 |
|---|---|
| 业务线排布软偏好 | 当前不实现，预留字段，未来启用 |
| 结果对比（多份历史） | 用完即弃不留痕，用户自行保存导出对比 |
| 锁定满意项目 | 与全局调度冲突，复杂度高 |
| CP-SAT/OR-Tools 精确求解 | MVP 贪心足够，M5 评估是否需要 |
| 内置静态节假日表 | 用户明确不维护 |
| 手动导入节假日 CSV | 用户明确不要 |
