# UC-export-001 汇总导出

## 参与者

- 单人全包用户

## 前置条件

- 已完成生成且合规校验通过（UC-audit-001）。

## 主流程

1. 用户在结果页选择"导出"。
2. 系统生成两份产物：
   - **CSV 原始数据**：列 = 项目、员工、日期、工时；每条 = 一员工一日期一项目的工时。
   - **Excel 汇总表**：维度 = 项目-员工；含日/周/月/年四种粒度汇总。
3. 用户下载两份文件（离线保管，工具不留存）。

## 异常流程

- **2a** 导出失败（磁盘权限/空间不足）：报错并提示原因。

## 后置条件

- 用户获得 CSV 与 Excel，可提交审计或进一步分析。
- 工具内不保留导出文件副本（ADR-0006 D1）。

## 设计影响

- **双产物固定输出**：CSV 与 Excel 同时生成，不强制勾选粒度。
- **Excel sheet 组织待定**：建议按粒度分 sheet（日/周/月/年各一），审计视角直观。→ 待 plan 确认。
- **汇总值与 CSV 一致性**：导出时程序化校验一致性。
- **导出即用户离线保管**：工具不承担归档职责。
