# 投入比例领域

> 分母、分子、周期、离散化、容差的领域定义与规则。

## 概念定义

### 投入目标比例（target ratio）
项目在某周期内应占用的工时比例。非绝对工时，是相对全员可用工时的比例。

### 周期（period）
投入比例的时间粒度，三种：
- 年（year）
- 季度（quarter）
- 月（month）

每个周期可独立表达比例。周期边界按自然年/季/月划分。

### 分母（denominator）
投入比例的分母 = 周期内[全员](staff.md#全员all-staff)可用工时 = 全员有效工作日 × 8h。
- 有效工作日 = 周期内日期 − [节假日](work-hours.md#节假日规则) − [年假日](work-hours.md#年假规则)，且员工在职。

### 分子（numerator）
投入比例的分子 = 项目在该周期内实际分配的工时。

### 离散化（discretization）
满载工作日是离散单位（每天 0 或 8h），比例只能取离散值 k/N。
- 不拆分时：步长 = 8h = 1 工作日，小规模场景步长过大。
- 1h 拆分时：步长 = 1h，分子可精确凑配（满载与精确比例不矛盾——满载约束每日总和，1h 拆分约束项目间分配）。

### 容差（tolerance）
生成结果比例与目标比例的允许偏差。1h 粒度下残余误差 ≤ 1h，容差可设很小。

## 规则

### 比例换算规则
- 每周期项目实际分配工时 ≈ 该周期全员可用工时 × target_ratio（容差内）。
- 分母 = 周期内全员有效工作日 × 8h。
- 分子 = 项目分得工时 = Σ 每个员工每天分给该项目的 1h 槽位数。

### 周期规则
- 项目声明周期粒度（年/季/月），每周期独立 target_ratio。
- 项目跨多周期时，各周期比例独立表达。
- 周期边界跨项目区间时，按日归属周期。
- 分母按周期内全员可用工时计算。

### 离散化规则
- 1h 拆分粒度让分子步长 = 1h，小规模场景可精确凑配。
- 满载与精确比例不矛盾：满载约束每日总和 = 8h，1h 拆分约束 8h 在项目间分配。

## 关系

- 分母依赖[人事](staff.md)（全员在职区间）与[工时](work-hours.md)（节假日、年假排除）。
- 分子依赖[项目](project.md)（分配工时给哪个项目）与工时（1h 拆分）。
- 离散化依赖工时的 1h 拆分粒度。
- 比例达成度是[合规](compliance.md)校验项之一。
