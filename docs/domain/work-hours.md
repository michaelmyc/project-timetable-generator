# 工时领域

> 工时、工作日、满载、拆分、节假日、年假的领域定义与规则。

## 概念定义

### 工时（work hours）
员工在某日期为某项目工作的小时数。基本记录单元：一员工一日期一项目的工时（[WorkHourRecord](glossary.md#工时work-hours)）。

### 工作日（workday）
分配了工时的日期。工作日每人当日总工时 = 8h（满载）。

### 非工作日
不分配工时的日期：
- 节假日（法定假日，含调休判断）
- 年假日
- 周末（降级模式下）

## 规则

### 满载规则
- 工作日每人当日总工时 **= 8h**（满载，非上限）。
- 不存在部分负载日（如 4h 天）——人一天工作就是满的。
- 非工作日工时 = 0h。

### 1h 拆分规则
- 满载工作日的 8h 可按 **1h 粒度**拆分给多个项目。
- 拆分示例：A 3h + B 5h = 8h；A 4h + B 4h = 8h；A 8h + B 0h = 8h（不拆分）。
- 同日跨项目工时总和 **= 8h**。
- 每个 (员工, 日期, 项目) 分配为 1h 整数倍 ∈ {0,1,...,8}。

### 节假日规则
- 法定节假日不分配工时。
- 调休工作日（周末补班）可分配工时。
- 节假日数据来源为外部 API（timor.tech），含 `is_workday` 标记区分节假日与调休工作日。
- API 不可用时降级为"仅周末排除"（周六日非工作日，不考虑法定节假日与调休）——降级是产品决策，领域事实是"节假日 = 法定假日 + 调休"。

### 年假规则
- 员工年假额度内休满，年假日不分配工时。
- 年假安排落在员工在职区间内且非节假日。
- 年假安排倾向分散（避免连续大段）——分散是实现策略，领域事实是"年假须休满且不分配工时"。

## 关系

- 工时依赖[项目](project.md)（分配给哪个项目）与[人事](staff.md)（哪个员工）。
- 节假日影响[比例](ratio.md)分母（有效工作日 = 区间 − 节假日 − 年假日）。
- 年假影响分母与[合规](compliance.md)（年假休满是硬约束）。
- 满载与拆分影响比例[离散化](ratio.md#离散化discretization)（1h 拆分细化步长）。
